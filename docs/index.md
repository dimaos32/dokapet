# Обзор

Это учебный pet-проект для изучения возможностей генератора статических сайтов [VitePress](https://vitepress.dev/).

Основная цель проекта - разобраться в подходах к созданию современной документации на базе Markdown, организации структуры контента, настройке навигации и возможностях кастомизации [VitePress](https://vitepress.dev/).

Почему именно [VitePress](https://vitepress.dev/):

- документация хранится в одном репозитории с проектом и развивается вместе с ним;
- при необходимости документацию можно вынести в отдельный репозиторий;
- [VitePress](https://vitepress.dev/) может быть установлен практически в любой существующий проект;
- документация собирается отдельно от основного приложения;
- для документации запускается собственный dev-сервер;
- для старта практически не требуется конфигурация — достаточно добавить Markdown-файлы;
- при этом [VitePress](https://vitepress.dev/) предоставляет широкие возможности для настройки внешнего вида, структуры и поведения сайта;
- поддерживаются кастомные темы, компоненты, Vue-интеграция, расширение Markdown и собственная логика;
- проект хорошо подходит как для небольших заметок и внутренних wiki, так и для полноценных публичных документационных порталов.

Проект используется исключительно в учебных целях для изучения возможностей [VitePress](https://vitepress.dev/) и связанных с ним подходов к организации документации.
